# 随叫随到

AstrBot 插件 —— 通过昵称或自定义名字唤醒机器人，无需 @。

## 功能

- 发 `{名字}今天天气怎么样` 即可唤醒机器人，无需 @mention
- 支持配置最多 3 个唤醒名字
- 未配置时自动检测机器人 QQ 昵称作为默认唤醒名
- 管理指令：`callme list` / `callme add` / `callme remove`
- WebUI 插件配置页也支持修改唤醒名字列表

## 安装

将 `astrbot_plugin_call_me` 文件夹放入 AstrBot 的 `data/plugins/` 目录，重启 AstrBot 即可。

## 使用

### 唤醒

配置好唤醒名字后（比如叫"小白"），在群聊中直接发送：

```
小白今天天气怎么样
```

机器人会自动响应，如同被 @ 一样。

### 管理

```
callme                    — 查看当前唤醒名字
callme list               — 列出唤醒名字
callme add <名字>         — 添加唤醒名字（最多 3 个）
callme remove <名字>      — 移除唤醒名字
callme help               — 查看帮助
```

也可以在 WebUI 的插件配置页中修改 `name_list`。

## 配置

| 字段 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `name_list` | `list` | `[]` | 唤醒名字列表（最多 3 个），留空则自动检测机器人昵称 |

## 原理

插件启动时将唤醒名字同步写入 AstrBot 的 `wake_prefix`，由框架原生处理名字匹配与前缀剥离，无需拦截消息管道。
